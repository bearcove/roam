# @bearcove/telex-postcard

Postcard-compatible serialization utilities for Telex TypeScript packages.

## Role in the Telex stack

`@bearcove/telex-postcard` provides binary encoding/decoding support used by wire and runtime layers.

## What this package provides

- Postcard-oriented schema codecs
- Primitive/value encoding helpers used by protocol message serialization

## Fits with

- `@bearcove/telex-wire` for full wire message encoding
- `@bearcove/telex-core` runtime call/response machinery

Part of the Telex workspace: <https://github.com/bearcove/telex>
