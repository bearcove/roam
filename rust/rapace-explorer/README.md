# rapace-explorer

[![crates.io](https://img.shields.io/crates/v/rapace-explorer.svg)](https://crates.io/crates/rapace-explorer)
[![documentation](https://docs.rs/rapace-explorer/badge.svg)](https://docs.rs/rapace-explorer)
[![MIT/Apache-2.0 licensed](https://img.shields.io/crates/l/rapace-explorer.svg)](./LICENSE)

Explorer service for rapace - dynamic service discovery and method invocation.

Provides a special service for runtime service introspection and dynamic RPC calls:

- **Service discovery**: List all registered services and their methods
- **Dynamic calls**: Call methods without compile-time type information
- **Type introspection**: Inspect method signatures and argument types
- **Web UI ready**: Powers the rapace dashboard and admin interfaces

Useful for tools, dashboards, and debugging applications.
